# 추천 광고 로직 고도화를 통한 사용자 경험 개선

> 2024.10 ~ 2024.11 · 문제 분석 주도 및 백엔드 개발 담당, 프론트엔드 개발자와 협업

`Java` `Spring Boot`

**검색 결과와 무관한 광고 노출 문제 해결**

<br/>

## Background

- 앱 스토어 리뷰를 통해 검색 결과와 관련 없는 광고가 노출된다는 사용자 불만 인입
- 트렌비는 검색 페이지와 카테고리 페이지가 동일한 구조로 되어 있음
- 사용자가 "루이비통 가방"으로 검색 후 "여성" 카테고리 필터를 적용하면, 광고 추천 시스템이 검색어 컨텍스트를 잃고 카테고리 필터 기준으로만 광고를 반환
- 결과적으로 검색 의도와 무관한 광고가 노출되고, 검색어 기반 결과가 사라지는 문제 발생

<br/>

## Approach

### 문제 흐름 분석

- 검색→필터 적용 시 광고 API로 전달되는 파라미터 흐름을 추적하여, 검색어 컨텍스트가 유실되는 지점을 특정

### API 파라미터 재설계

- 프론트엔드 개발자와 협의하여 검색어와 필터 값을 동시에 유지하는 API 파라미터 구조를 새롭게 정의
- 백엔드 추천 로직에서 검색어 + 필터 조건을 모두 반영하여 광고를 선별 노출하도록 변경

<br/>

## Results

- 검색 필터 적용 시에도 검색 의도에 부합하는 광고만 노출되도록 개선
- 앱 스토어 부정 리뷰로 접수된 사용자 불만 해소
