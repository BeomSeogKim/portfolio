# 추천 광고 로직 고도화를 통한 사용자 경험 개선

> 2024.10 ~ 2024.11

`Java` `Spring Boot`

**검색 결과와 무관한 광고 노출 문제 해결**

<br/>

## Background

- 앱 스토어 리뷰를 통해 검색 결과와 관련 없는 광고가 노출된다는 사용자 불만 인입
- 원인 분석 결과, 검색 필터와 광고 추천 시스템 간의 상태 값 불일치가 근본 원인

<br/>

## Approach

- Frontend 팀원과 논의 후, 사용자의 검색 컨텍스트(필터 값 등)를 유지하는 API 파라미터를 새롭게 정의
- 확장된 파라미터를 활용하여 사용자의 필터 의도에 맞는 광고만 선별 노출하도록 추천 로직 변경

<br/>

## Results

- 검색 필터와 무관한 광고 노출 문제 해결
