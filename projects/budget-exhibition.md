# 초저가 기획전 개발

> 2024.06 ~ 2024.07

`Java` `Spring Boot` `MySQL`

**구매 전환율 40% 성장, 운영 리소스 월 20시간 이상 수작업 100% 자동화**

<br/>

## Background

- MD가 수작업으로 상품 카탈로그 번호를 입력하여 저가 상품들을 진열시켜 기획전을 운영

<br/>

## Approach

- 실시간 상품 진열을 위해 별도의 위젯 생성
- CronWorkflow를 활용하여 12시간 주기로 초저가 뱃지가 부여된 상품을 자동 업데이트하는 프로세스 구축
- 자동화 이후 프로모션 상품 수가 급증하며 MySQL에서 `Packet for query is too large` 오류 발생
- 동료 개발자, 기획자와의 논의를 통해 경쟁력이 높은 1,000개의 상품만 전시하는 방향으로 문제 해결

<br/>

## Results

- 구매 전환율 40% 성장
- 월 20시간 이상 소요되던 MD 수작업을 100% 자동화
